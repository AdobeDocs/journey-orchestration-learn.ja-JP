# 別のジャーニーへのジャンプ

個人を別のジャーニーに移動させる方法を説明します。

ジャンプアクションアクティビティを使用すると、次のことができます。

* 非常に複雑なジャーニーを複数のジャーニーに分けて設計を簡素化する
* 一般的で再利用可能なジャーニーパターンに基づいてジャーニーを作成する
