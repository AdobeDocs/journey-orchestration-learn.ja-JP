# 別のジャーニーにジャンプ

個人を別のジャーニーに移動させる方法を説明します。

ジャンプアクションアクティビティでは、次の操作を行うことができます。

* 非常に複雑なジャーニーを複数に分割して設計を簡素化
* 共通の再利用可能なジャーニーパターンに基づくジャーニーの構築
